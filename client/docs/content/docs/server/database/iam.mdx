## 4. Access Control (RBAC)

### `roles`
Stores access role classes.
* **Schema**: `id` (UUID), `name` (unique: `'admin'`, `'user'`), `description`, `created_at`.

### `permissions`
Exposes resources and action limits for security mapping.
* **Schema**: `id` (UUID), `resource` (e.g. `'roles'`), `action` (e.g. `'read'`, `'create'`), `created_at`.
* **Unique Constraints**: `(resource, action)` ensures resource actions remain unique.

### `role_permissions`
Junction table mapping permissions to roles.
* **Schema**: `role_id` (foreign key roles), `permission_id` (foreign key permissions). Joint primary key.

### `user_roles`
Junction table mapping users to roles.
* **Schema**: `user_id` (foreign key users), `role_id` (foreign key roles), `assigned_at`. Joint primary key.
