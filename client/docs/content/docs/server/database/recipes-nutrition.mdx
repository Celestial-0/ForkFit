## 1. Food Catalog & Costs

### `food_items`
Defines base food items and their macronutrient distributions per 100 grams.

* **Schema**:
  | Column | Type | Constraints | Default | Description |
  | :--- | :--- | :--- | :--- | :--- |
  | `id` | `uuid` | `PRIMARY KEY` | `gen_random_uuid()` | Food item identifier. |
  | `name` | `text` | `NOT NULL UNIQUE` | - | Descriptive food name. |
  | `description` | `text` | - | - | Food item details. |
  | `calories_per_100g` | `numeric(6,2)` | `NOT NULL` | - | Calorie count per 100g. |
  | `protein_per_100g` | `numeric(5,2)` | `NOT NULL` | - | Protein content (g) per 100g. |
  | `carbs_per_100g` | `numeric(5,2)` | `NOT NULL` | - | Carbohydrates content (g) per 100g. |
  | `fat_per_100g` | `numeric(5,2)` | `NOT NULL` | - | Fat content (g) per 100g. |
  | `fiber_per_100g` | `numeric(5,2)` | `NOT NULL` | `0.00` | Dietary fiber (g) per 100g. |
  | `sodium_mg_per_100g` | `numeric(10,2)` | `NOT NULL` | `0.00` | Sodium level (mg) per 100g. |
  | `micronutrients` | `jsonb` | `NOT NULL` | `'{}'` | Key-value store for vitamins and minerals. |
  | `barcode` | `text` | - | - | UPC/EAN barcode string. |
  | `is_verified` | `boolean` | `NOT NULL` | `false` | True if verified by nutrition experts. |
  | `food_code` | `text` | `UNIQUE` | - | Internal/standard system code. |
  | `primary_source` | `text` | - | - | Origin source registry (e.g. `'Anuvaad INDB'`). |
  | `raw_food_cost_id` | `uuid` | `REFERENCES raw_food_costs(id) ON DELETE SET NULL` | - | Optional link to raw food cost pattern. |
  | `created_at` | `timestamptz` | `NOT NULL` | `now()` | Record creation timestamp. |
  | `updated_at` | `timestamptz` | `NOT NULL` | `now()` | Record change timestamp. |

### `raw_food_costs`
Defines price guidelines per 100 grams for generic food patterns in India (utilized by the Budget Agent for dynamic cost evaluation).

* **Schema**:
  | Column | Type | Constraints | Default | Description |
  | :--- | :--- | :--- | :--- | :--- |
  | `id` | `uuid` | `PRIMARY KEY` | `gen_random_uuid()` | Cost pattern identifier. |
  | `food_pattern` | `text` | `NOT NULL UNIQUE` | - | Pattern for name matching (e.g. `'chicken'`). |
  | `cost_per_100g` | `numeric(10,2)` | `NOT NULL` | - | Price per 100 grams in India. |
  | `price_currency` | `text` | `NOT NULL` | `'INR'` | Price currency code. |
  | `created_at` | `timestamptz` | `NOT NULL` | `now()` | Record creation timestamp. |
  | `updated_at` | `timestamptz` | `NOT NULL` | `now()` | Record change timestamp. |

### `food_item_portions`
Specifies portion equivalence factors to convert serving units (e.g., cups, slices) into grams.

* **Schema**: `id` (PRIMARY KEY), `food_item_id` (foreign key food_items), `serving_unit` (e.g. `'cup'`), `grams_equivalent` (checking `> 0`), `created_at`, `updated_at`. Joint unique index on `(food_item_id, serving_unit)`.

## 2. Recipe Structure

### `recipes`
Header metadata for food recipes.

* **Schema**:
  | Column | Type | Constraints | Default | Description |
  | :--- | :--- | :--- | :--- | :--- |
  | `id` | `uuid` | `PRIMARY KEY` | `gen_random_uuid()` | Unique recipe identifier. |
  | `owner_id` | `uuid` | `REFERENCES users(id) ON DELETE SET NULL` | - | User owner. Null indicates a public system recipe. |
  | `parent_recipe_id` | `uuid` | `REFERENCES recipes(id) ON DELETE SET NULL` | - | Reference to parent recipe (for alternatives/variations). |
  | `title` | `text` | `NOT NULL` | - | Recipe title. |
  | `description` | `text` | - | - | Recipe brief description. |
  | `instructions` | `text[]` | `NOT NULL` | - | Array list of recipe execution steps. |
  | `prep_time_minutes` | `integer` | - | - | Preparation time in minutes. |
  | `cook_time_minutes` | `integer` | - | - | Cooking duration in minutes. |
  | `servings` | `numeric(4,2)` | `NOT NULL` | `1.00` | Default servings yield. |
  | `cuisine` | `text` | - | - | Regional classification (e.g. `'North Indian'`). |
  | `course` | `text` | - | - | Course category (e.g. `'Lunch'`, `'Breakfast'`). |
  | `dietary_tags` | `text[]` | `NOT NULL` | `'{}'` | Dietary categories (e.g. `{'Vegan', 'Gluten-Free'}`). |
  | `source_url` | `text` | - | - | Original recipe URL source link. |
  | `is_public` | `boolean` | `NOT NULL` | `false` | True if visible to other platform users. |

### `recipe_food_items`
Junction table tracking recipe food item requirements.

* **Schema**: `recipe_id` (foreign key recipes), `food_item_id` (foreign key food_items), `quantity` (numeric), `unit` (e.g. `'g'`), `grams_equivalent` (computed weight), `notes`. Joint primary key on `(recipe_id, food_item_id)`.
