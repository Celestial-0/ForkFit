## 3. Shopping Lists

### `shopping_lists`
Shopping lists generated from meal programs.
* **Schema**: `id` (PRIMARY KEY), `user_id` (foreign key users), `name` (list title), `status` (default `'active'`, can be `'completed'`, `'archived'`), `created_at`, `updated_at`.

### `shopping_list_items`
Detailed elements on a shopping list.
* **Schema**: `id` (PRIMARY KEY), `shopping_list_id` (foreign key shopping_lists), `ingredient_id` (foreign key ingredients), `custom_item_name`, `quantity`, `unit`, `is_acquired` (default `false`), `category` (default `'Other'`, e.g., `'Produce'`), `created_at`, `updated_at`.
