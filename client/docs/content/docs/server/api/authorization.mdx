# RBAC & Authorization 

ForkFit supports robust, granular Role-Based Access Control (RBAC). Admin roles are seeded automatically at startup.

---

## 1. Role Management

### List Roles

`GET /api/v1/auth/roles`

Returns all configured security roles.

---

### Create Custom Role

`POST /api/v1/auth/roles`

Creates a new access role.

* **Payload Parameters**:
  <TypeTable
    type={{
      name: { type: 'string', required: true, description: 'Unique name of the role (e.g. trainer).' },
      description: { type: 'string', description: 'Brief context description.' }
    }}
  />

---

## 2. Permissions & Assignments

### Map Permission to Role

`POST /api/v1/auth/roles/{role_id}/permissions/{permission_id}`

Assigns a permission code to a specified role.

---

### Assign Role to User

`POST /api/v1/auth/users/{user_id}/roles/{role_id}`

Grants a role to a user.
