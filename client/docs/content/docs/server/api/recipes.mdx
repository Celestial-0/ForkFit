# Recipes & Nutrition 

Endpoints managing the food catalog, recipe creation, and nutritional logging.

---

## 1. Food Items Queries

### Search Food Items Database

`GET /api/v1/food-items/search`

Returns matching food items from our nutritional catalog.

* **Query Parameters**:
  <TypeTable
    type={{
      q: { type: 'string', required: true, description: 'Food item search query.' }
    }}
  />

---

### Get Food Item Details

`GET /api/v1/food-items/{id}`

Returns macro details (protein, fat, carbs, calories) for a specific food item ID.

---

## 2. Recipes & Logging

### Create Recipe

`POST /api/v1/recipes`

Saves a custom recipe consisting of multiple food items.

---

### Log Food Consumption

`POST /api/v1/food-logs`

Logs a food item or recipe consumed in a specific portion size.

---

### Get Daily Macros

`GET /api/v1/food-logs/daily`

Returns a summarized breakdown of protein, fat, carbs, and calories consumed today vs user targets.
