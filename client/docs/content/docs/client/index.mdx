# Client Documentation

Welcome to the client-side documentation for the ForkFit application suite. 

ForkFit provides native and web client experiences built to interact seamlessly with our core server platform:

* **Web Application**: A high-performance web experience built with React, Next.js, and TypeScript.
* **Native Mobile Application**: An iOS and Android mobile app designed for on-the-go meal logging, workout tracking, and live AI assistant coaching.

Select a section from the sidebar or search to get started.
