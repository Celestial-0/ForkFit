## Core Features

Our platform is designed with security and developer experience at its core:

* **OAuth 2.0 Integration**: Built-in support for Google and GitHub authentication.
* **Secure Session Control**: Comprehensive tools to list, inspect, and revoke active device sessions.
* **OTP Verification**: Multi-layered security using time-sensitive One-Time Passwords for email verification and password resets.
* **Granular RBAC**: Fully customizable roles and permission mappings to enforce precise access levels across endpoints.
* **Rate Limiting**: Built-in protection against DDoS and abuse using token bucket algorithm controls.

<Callout type="info" title="Looking for Client SDKs?">
  Official SDKs for JavaScript/TypeScript, Python, and Go are currently in development. For now, you can use standard HTTP client libraries (like `axios`, `fetch`, or `requests`) to interact with the raw HTTP API.
</Callout>
